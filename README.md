<h1>Сервер для обнаружения - есть ли на снимке разлив нефти</h1>
<h4>Реализованная функциональность</h4>
<ul>
 <li>Информация о наличии разива нефти на снимке</li>
</ul> 
<h4>Особенность проекта в следующем:</h4>
<ul>
 <li>комплексный подход с созданием инструмента, предоставляющего проработанные аспекты UI с мощными методами машинного обучения "под капотом"</li>
</ul>
<h4>Основной стек технологий:</h4>
<ul>
  <li>Python</li>
  <li>Flask</li>
  <li>SKLearn</li>
  <li>Docker</li>
</ul>
<h4>Демо</h4>
<p>Docker-имидж демо приложения для android доступно по адресу: https://hub.docker.com/repository/docker/vitek999/oil_spill_detector</p>
<p>Снимки берутся из базы предоставленной sentinel-hub https://www.sentinel-hub.com/ </p>
<p>Для получения информаци о наличии нефти на снимке необходимо сделать GET-запрос с указанием координат и промежутка дат снимка Пример запроса: <code>
http://localhost:8080/predict?height=64&width=64&bbox=73.03322551561014,60.588367153736506,73.04455362140644,60.59428130212325&date_start=2020-06-01&date_end=2020-06-30
</code></p>
